Мой пет проект на Spring Boot + React.js. Суть проекта: сайт для обучения программированию у менторов.
Навскидку функционал будет разделен между 4 группами пользователей: менторы, ученики, модераторы, администратор.
Ученик может выбрать себе ментора, пользуясь информацией из его профиля, отзывах и т.д.
Оплата резервируется в системе до потдвеждения клиентом, или после n дней после оказания услуги. Спорные случаи
решаются через арбитраж.
Для общения используется встроенный месседжер, а также через видеоконференции. Ментор выдаёт дз, ученик сдаёт их.
Модераторы отвечают за решение спорных вопросов, то есть систему арбитража. Также они занимаются любыми другими
сообщениями в техподдержку.
Администратор - статистика по сайту, ключевые настройки, управление аккаунтами модераторов.
